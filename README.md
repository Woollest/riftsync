# RiftSync

RiftSync は、League of Legends のチャンピオンセレクト中に使えるピック支援Webアプリです。

味方が選んだチャンピオンを1体入力すると、自分のロールで相性の良い候補を3体表示します。初心者でも短時間で判断しやすいように、アイコン、星評価、コンボ相性、勝率、試合数、短い理由を中心にまとめています。

[RiftSyncを開く](https://woollest.github.io/riftsync/)

## RiftSyncでできること

- 味方1体に合わせたおすすめピックを3体確認できます
- おすすめ上位が使えない時のために、追加候補も確認できます
- 勝率が低めの「非推奨候補」を必要な時だけ開けます
- OP.GG直接データか、推定補完かを見分けられます
- 試合数や信頼度を見ながら、データを過信しすぎず判断できます
- 結果テキストや選択条件リンクを友人に共有できます
- スマホではホーム画面に追加して、アプリのように起動できます

## こんな時に便利です

- ソロQのチャンピオンセレクトで、味方に合わせたピックを選びたい
- 「このADCならどのサポートが合う？」をすぐ見たい
- 自分のロールだけ決まっていて、何を出すか迷っている
- 勝率だけでなく、組み合わせとしてかみ合うかも見たい
- 友人に「この組み合わせどう？」を共有したい
- 難しいチャンピオンかどうかも一緒に見たい

## 使い方

1. 味方のロールを選びます
2. 味方のチャンピオンを選びます
3. 自分のロールを選びます

選択すると、おすすめ3体が自動で表示されます。スマホでは縦に見やすく、PCでは左に入力、右に結果が出る横長レイアウトになります。

味方チャンピオン一覧は、選んだ味方ロールに合いやすいチャンピオンを上側にまとめ、その下に残りのチャンピオンを表示します。オフメタの候補も選べるので、ADCブランドのような珍しいピックにも対応しやすくしています。

## おすすめカードの見方

- 総合評価: 星5段階。コンボ相性、勝率、メタ評価をまとめた目安です
- コンボ相性: 味方チャンピオンとの組み合わせ評価です
- 勝率: OP.GGのシナジーデータまたはロール統計に基づく勝率です
- 試合数: その判断に使ったデータ量です
- 信頼度: 直接相性の有無と試合数から、高 / 中 / 低で表示します
- OP.GG直接: OP.GGの個別シナジーページ由来の組み合わせです
- 推定補完: 直接データが少ない時に、ロール統計やチャンピオン性質から補っています
- メタ: 炎マークで現在の強さをざっくり表示します
- 難易度: Riot公式難易度を「簡単 / 普通 / 難しい」に変換しています
- 初心者おすすめ: 難易度が簡単なチャンピオンに表示します
- データ少: 試合数が少なく、判断に注意したい候補に表示します

おすすめ理由は1行だけにしています。チャンピオンセレクト中でも、数値と理由を合わせてすぐ判断できるようにするためです。

## 判断のコツ

まずは「OP.GG直接」と「信頼度」を見てください。直接データがあり、試合数も十分ある候補は比較的参考にしやすいです。

次に、コンボ相性と勝率を見ます。コンボ相性が高い候補は味方の強みと合わせやすく、勝率が高い候補は統計上の結果が良い傾向にあります。

最後に、自分が使えるかを確認してください。総合評価が高くても、操作が難しい、練習していない、レーンで苦手という場合は、追加候補から選ぶのも良い判断です。

## 共有機能

結果画面には、友人と相談しやすいように共有機能があります。

- 結果をコピー: おすすめ3体、コンボ相性、勝率、信頼度をテキストでコピーできます
- リンクをコピー: 味方ロール、味方チャンピオン、自分ロールを再現できるURLをコピーできます
- フィードバック: 気づいた点をGitHub Issuesへ送れます

リンクを共有すると、相手も同じ条件のおすすめ結果を開けます。

## スマホでアプリのように使う

スマホでは、ブラウザのメニューから「ホーム画面に追加」を使うと、RiftSyncをアプリのように起動できます。

ホーム画面追加後は、チャンピオンセレクト中にすぐ開きやすくなります。表示が古い、アイコンが反映されない、画面が崩れるなどがある場合は、一度ブラウザで開き直してから再追加してみてください。

## データについて

RiftSyncは、OP.GGのGlobal / Gold+ / Ranked Solo/Duoの統計と、Riot Data Dragonのチャンピオン情報を使っています。

アプリ内の「データ情報」には、パッチ、更新日、出典、ランク帯、地域、相性データ件数、直接相性件数、採用方式が表示されます。表示中の候補がどれくらい信頼できそうかを見る時は、勝率だけでなく信頼度と試合数も一緒に確認してください。

直接相性データが少ない組み合わせでは、推定補完の候補も表示されます。これは候補を空にしないための仕組みですが、直接データよりは慎重に見てください。

## よくある質問

### 敵のピックやBANは考慮されますか？

現時点では考慮しません。データの信頼性を保つため、入力は味方1体と自分のロールに絞っています。

### 味方を2体以上入力できますか？

現時点では1体のみです。入力数を増やすと組み合わせごとの試合数が少なくなり、統計の信頼性が落ちやすいためです。

### おすすめに知らないチャンピオンが出たらどうすればいいですか？

無理に選ばなくて大丈夫です。追加候補や非推奨候補も見ながら、自分が使えるチャンピオンを優先してください。

### 「推定補完」は悪い候補ですか？

悪い候補という意味ではありません。OP.GGの直接シナジーデータがない、または少ない時に、ロール統計やチャンピオン性質から補っている候補です。信頼度や試合数と一緒に見てください。

### どのランク帯のデータですか？

基本は OP.GG の Global / Gold+ / Ranked Solo/Duo を元にしています。正確なパッチや更新日は、アプリ内の「データ情報」で確認できます。

## 注意
