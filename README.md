# RiftSync

RiftSync は、League of Legends のチャンピオンセレクト中に使うピック支援Webアプリです。

味方が選んだチャンピオンを1体入力すると、自分のロールで相性の良い候補を3体表示します。初心者でも短時間で判断しやすいように、アイコン、星評価、勝率、試合数、短い理由を中心にまとめています。

[RiftSyncを開く](https://woollest.github.io/riftsync/)

## こんな時に

- ソロQのチャンピオンセレクトで、味方に合わせたピックを選びたい
- 「このADCならどのサポートが合う？」をすぐ見たい
- 使いやすさだけでなく、ペア勝率や試合数も見て判断したい
- 友人におすすめ候補を共有したい
- 初心者向けの難易度表示も一緒に見たい

## 使い方

1. 味方のロールを選びます
2. 味方のチャンピオンを選びます
3. 自分のロールを選びます

選択すると、おすすめ3体が自動で表示されます。スマホでは縦に見やすく、PCでは左に入力、右に結果が出る横長レイアウトになります。

## おすすめカードの見方

- 総合評価: 星5段階。味方との相性、勝率、メタ評価をまとめた目安です
- コンボ相性: 味方チャンピオンとの組み合わせ評価です
- 勝率: OP.GGのシナジーデータまたはロール統計に基づく勝率です
- メタ: 炎マークで現在の強さをざっくり表示します
- 難易度: Riot公式難易度を「簡単 / 普通 / 難しい」に変換しています
- 試合数: その判断に使ったデータ量です。少ない場合は「データ少」と表示します

「OP.GG上位」と書かれている候補は、OP.GGの個別シナジーページで上位に出ている組み合わせを元にしています。

## 便利な機能

- 結果をコピー: おすすめ3体をテキストでコピーできます
- リンクをコピー: 選択中の条件をそのまま開けるURLをコピーできます
- 非推奨候補: 勝率が低めの候補を折りたたみで確認できます
- フィードバック: 気づいた点をGitHub Issuesへ送れます

## データについて

RiftSyncは、OP.GGのGlobal / Gold+ / Ranked Solo/Duoの統計と、Riot Data Dragonのチャンピオン情報を使っています。

アプリ内の「データ情報」には、パッチ、更新日、出典、相性データ件数、直接相性件数が表示されます。表示中の候補がどれくらい信頼できそうかを見る時は、勝率だけでなく試合数も一緒に確認してください。

## 注意

RiftSyncはピック判断を助けるためのツールです。敵のピック、BAN状況、自分の熟練度、チーム全体の作戦までは完全には反映しません。最後は自分が使えるチャンピオンかどうかも含めて選んでください。
